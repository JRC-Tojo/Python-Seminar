# ML-Seminar
2022年度の機械学習ゼミにおける問題とその解答解説



# 概要

2022年度は機械学習ゼミの中で[機械学習帳](https://chokkan.github.io/mlnote/index.html)と追加課題の両者を出題することで，Pythonによるプログラミングスキル向上とともに，機械学習に対する基礎的な理解を得ることを目的とした．

全6回で構成されており，後半になるほど複雑，かつ応用的な考え方を使う必要があるようになっている．

出題には[scrapbox](https://scrapbox.io/actscape-seminar/%E6%A9%9F%E6%A2%B0%E5%AD%A6%E7%BF%92%E3%82%BC%E3%83%9F)を用いており，本リポジトリはその答え合わせや復習を目的として活用することを想定している．

答えをそのまま列挙するため，出題時にはscrapboxの活用などを通じた配慮が必要である．

基本的な出題方針は「自分で調べて目標とする処理を実装できる」ことにあるため，与えられた問題に対して関連する知識をGoogleなどで検索して実装してみること．

一部にはゼミの復習も含んでいるため，全ての問題に取り組むことで，本ゼミが目標とするスキルをすべて身に着けることが可能である．



# 各プログラムの構成

各プログラムは`.py`形式のPythonファイルであり，Jupyterに対応した`.ipynb`形式にはなっていない．

強く推奨する記法として，以下のようなクラスや関数部分と，実行部分を分けて書くものがある．

これにより，プログラム全体の見通しが良くなり，いわゆるツール部分である関数やクラスの定義と，ツールの活用部分である実装を分離した記述が可能

ゼミの課題のみならず，実際に研究でプログラムを書く時にもこのように記載しておくと便利

